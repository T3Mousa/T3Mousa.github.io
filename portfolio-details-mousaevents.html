Mousa Events

Mousa Events is an informational website for a client with an event planning business. The site was created based on the specifications of the client and includes a user inquiry contact form. Mousa Events is created using JavaScript, Node, Express, React, HTML and CSS, and is hosted on Render.com and linked to the clients' unique domain name.
